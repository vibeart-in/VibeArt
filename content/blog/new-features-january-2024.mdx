## üîß Developer Features

New tools for developers and power users.

### API v2

Our new API is more powerful and easier to use:

```javascript
// New API v2 syntax
import { VibeArtClient } from "@vibeart/sdk";

const client = new VibeArtClient({
  apiKey: process.env.VIBEART_API_KEY,
});

// Generate an image
const result = await client.generate({
  prompt: "A serene mountain landscape",
  model: "sdxl-turbo",
  size: "1024x1024",
  steps: 4,
});

// Download the result
await result.download("output.png");
```

### Webhooks

Get notified when generation completes:

```typescript
interface WebhookPayload {
  event: "generation.completed" | "generation.failed";
  imageId: string;
  url: string;
  metadata: {
    prompt: string;
    model: string;
    duration: number;
  };
}

// Register a webhook
await client.webhooks.create({
  url: "https://your-app.com/webhook",
  events: ["generation.completed"],
});
```

### CLI Tool

New command-line interface for automation:

```bash
# Install the CLI
npm install -g @vibeart/cli

# Generate an image
vibeart generate "sunset over mountains" --model sdxl --output sunset.png

# Batch process
vibeart batch process ./images --upscale 2x --enhance

# Train a custom model
vibeart train --name my-style --images ./training-data
```
