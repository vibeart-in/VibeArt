Prompt engineering is both an art and a science. In this comprehensive guide, we'll explore advanced
techniques that will take your AI-generated artwork to the next level.

## Understanding Prompt Structure

The way you structure your prompts significantly impacts the quality of generated images. Let's
break down the anatomy of an effective prompt.

### Basic Prompt Components

Every great prompt consists of several key elements:

1. **Subject**: What is the main focus of the image?
2. **Style**: What artistic style should be applied?
3. **Mood**: What emotion or atmosphere should it convey?
4. **Technical Details**: Lighting, composition, camera angles

<Callout type="info" title="Quick Tip">
  Think of your prompt as a recipe. Each ingredient (component) contributes to the final result.
</Callout>

### The Power of Specificity

Compare these two prompts:

```
Basic: "A cat"
Advanced: "A fluffy Persian cat with blue eyes, sitting on a velvet cushion,
soft natural lighting, shallow depth of field, professional pet photography"
```

The difference in results is dramatic. Specificity gives the AI more context to work with.

## Advanced Prompt Techniques

Now let's dive into techniques used by professional AI artists.

### Weighted Keywords

You can emphasize certain elements by using weights:

```typescript
// Example of weighted prompt syntax
const prompt = {
  subject: "portrait of a woman",
  emphasis: {
    "detailed eyes": 1.5,
    "soft lighting": 1.3,
    photorealistic: 1.4,
  },
};
```

### Negative Prompting Strategy

Negative prompts are just as important as positive ones. Here's a strategic approach:

<Callout type="warning" title="Common Pitfall">
  Don't just list random negative terms. Be strategic about what you want to exclude.
</Callout>

**Categories of negative prompts:**

- **Quality issues**: blurry, low quality, pixelated, distorted
- **Unwanted elements**: text, watermark, signature, frame
- **Anatomical problems**: extra fingers, deformed, mutated
- **Style conflicts**: cartoon, anime (if going for realism)

### Style Mixing Techniques

Combining multiple artistic styles creates unique results:

```python
# Example style combinations
styles = [
    "impressionist + cyberpunk",
    "art nouveau + futuristic",
    "watercolor + geometric abstract"
]

for style in styles:
    generate_image(prompt=f"landscape in {style} style")
```

## Common Mistakes to Avoid

Learn from these frequent errors.

### Mistake 1: Overloading the Prompt

**Bad:**

```
Beautiful amazing stunning gorgeous incredible fantastic wonderful
perfect flawless masterpiece...
```

**Good:**

```
High quality, detailed, professional
```

Quality adjectives have diminishing returns. Use them sparingly.

### Mistake 2: Conflicting Instructions

Avoid contradictory elements:

- "Photorealistic anime" (style conflict)
- "Bright dark scene" (lighting conflict)
- "Minimalist highly detailed" (complexity conflict)

### Mistake 3: Ignoring Composition

Don't forget to specify composition:

```typescript
// Include composition guidance
const compositionTerms = [
  "rule of thirds",
  "centered composition",
  "leading lines",
  "symmetrical",
  "dynamic angle",
];
```
